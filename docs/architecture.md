### 📂 模組分層說明

| 資料夾       | 功能描述 |
|--------------|----------|
| `core/`       | 包含 Q-learning、策略判斷等非 UI 的邏輯 |
| `data/`       | 所有資料處理（CSV 載入、特徵建構、分析模組） |
| `controllers/` | 中介層，UI 不再直接控制學習與邏輯模組 |
| `ui/`         | 所有互動與畫面模組 |
| ...           | 其餘如 `scripts/`, `tests/`, `docs/` |
